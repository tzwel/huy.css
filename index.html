Attributes

Usage:

<element attribute> ... </element>
  • owl - adds margin-top to children as specified in the config.css
  • noowl - removes margin-top from a specific element as specified in the config.css
  • padding, padded - sets padding defined in the config.css
  • block - sets display to block
  • inline-block - sets display to inline-block
  • absolute - sets position to absolute
  • flex - sets position to flex

More will be documented soon, you can find all in attributes.css
